Capitulo 4: Fondos y Bordes

Los fondos y los bordes aportan soluciones para muchas cuestiones creativas de estilo en CSS como: imágenes de fondo, redondear esquinas o crear degradados

Estilo a los fondos: background

Background es una propiedad abreviada que se usa para los fondos en las cajas de CSS, con otras propiedades complejas.

Background-color

Define el color de fondo de cualquier elemento/caja en CSS, el color se va a aplicar por debajo del contenido y del padding. Los valores de color pueden ser todos los que permite esta propiedad.

Background-image

Permite poner una imagen de fondo a un elemento o caja (lo mas utilizado). De forma predeterminada una imagen grande no se ajusta a la caja, mientras que una imagen pequeña se repite hasta llenar la caja. Además si le aplicamos un color de fondo a la imagen, esta va a quedar por encima del color.

Background-repeat

Se usa para controlar el comportamiento repetitivo de las imágenes de fondo. Los valores son: no-repeat (evita que el fondo se repita), repeat-x (repite el fondo horizontalmente), repeat-y (repite el fondo verticalmente), repeat (repite el fondo de forma vertical y horizontal).

Background-size

Se usa para dimensionar las imágenes de fondo, usa tipo de valores de medida como longitud (px) y porcentaje (%). También tiene valores de palabra clave como: cover (el navegador agrandará la imagen para que cubra toda la caja, sin que se pierda su relación de aspecto, que se recorte la imagen). Por eso es probable que parte de la caja quede sin cubrir. También está contain: el navegador hará que la imagen cubra toda la caja a costa de cortar la imagen, perdiendo su relación de aspecto.

Background-position

Permite elegir la posición en la que la imagen de fondo aparece, usa el sistema de coordenadas x (horizontal: valor positivo mueve hacia la derecha y negativo hacia la izquierda) e y (vertical: valor positivo mueve hacia abajo y negativo hacia la izquierda). El valor predeterminado es (0,0) que se escribe 0% 0% que se ubica en la esquina superior izquierda.

Valores de position

Las palabras claves: top, right, left y bottom acomodan la imagen en el centro de estos lados. La palabra clave center pone la imagen en el centro de los cuatro lados.

También hay equivalencias entre valores los distintos tipos de valores: 50% es igual a center. Aunque en general se usan solo dos valores, uno para el eje x (mueve hacia derecha) y otro para el eje y (mueve hacia abajo), si los valores son negativos hacen lo contrario.

Gradient: degradado

Es un tipo de imagen que se puede usar como fondo, se establece usando background-image con el valor linear-gradient o radial-gradient

Para lograr un degradado profesional se utilizan tres colores que combinen, se les baja el tono a alguno (para evitar una linea oscura que une los colores), dos colores no dominantes y uno dominante (como se ven el las imágenes). Se puede usar para generar el código más fácil.

También podemos cambiar la posición del degradado rotando el grado del degradado lineal, no funciona para el degradado radial.

Multiples imágenes de fondo

Si hay varias imágenes de fondo se van a superponer una atrás de la otra, para evitar esto se puede usar background-position para mover las que están en el fondo (todas menos la primera). Las propiedades de background que estábamos viendo, también se pueden usar varias veces para que de tal forma van a afectar a las siguientes imágenes. Se pueden poner separadas por una coma, como se ve en la imagen; el primer grupo de valores afecta a la primera imagen y el segundo grupo corresponde a la segunda imagen, así sucesivamente. Cuando hay mas elementos que propiedades que determinan comportamientos, estas adquieren un ciclo: cuando se terminan, el primero vuelve a afectar al elemento que sigue.

Background-attachment: anclaje de fondo

Determina si la imagen de fondo se mueve cuando nos desplazamos por el contenido o queda fija.

Entran en función la barra de desplazamiento para el contenido y para la imagen de fondo.

Valores: Scroll cuando desplazas la imagen de fondo también se mueve el contenido. Fixed: la imagen de fondo no se extiende con el contenido ("se recorta"), la barra de desplazamiento de la imagen también mueva el contenido como en scroll. Local: el desplazamiento de la barra de contenido también mueve la imagen de fondo (se ve molesto). Combinaciones de valores: local y scroll si hay dos imágenes, el desplazamiento por el contenido mueve la primera imagen de fondo. Scroll y local: el desplazamiento por el contenido va a mover la ultima imagen de fondo.

Background

Es la propiedad abreviada para usar todas las propiedades anteriores en una sola linea, aunque tienen sus condiciones. Para añadir varios fondos, primero hay que definir todas las propiedades (separadas con un espacio) para el primer fondo y luego después de una coma podemos agregar otro fondo con todas sus propiedades.

Accesibilidad en los fondos

Primero debe haber diferencias de colores entre el fondo y el contenido de texto o lo que sea, para que se puede leer el contenido sin dificultades. También debes agregar un fondo por si la imagen no se carga y se puede leer el texto (usando la cascada escribimos primero el fondo alternativo y después el fondo "verdadero" para que lo reemplacé cuando se cargue ???).

Bordes

Por lo general se usa la propiedad abreviada borde para poner el ancho, estilo y color en una sola linea. Pero para diseños más creativos se puede usar cada lado del borde de manera diferente.

Propiedades abreviadas: border; border-width, border-style, border-color. Estas propiedades actúan sobre todos los lados de la misma manera.

Propiedades individuales: border-top, border-right, border-bottom, border-left. Para ser más especifico podemos usar: border-top-width, border-top-style y border-top-color, así con los demás lados. Estas propiedades tiene condiciones lógicas relacionadas con el tipo de escritura que se vera más adelante

Border-radius: esquinas redondeadas

Aplica sobre la esquina de la caja, distinto a border que solo aplica a los lados. Pueden usarse valores de longitud o porcentuales. Si ponemos dos valores: el primer valor define el radio horizontal (modifica la esquina 2 y 4, siguiendo la aguja del reloj), el segundo valor define el radio vertical (modifica la esquina 1 y 3); se usan para diseños creativos. Para diseños más convencionales se usa un valor que aplica estilo a las cuatro esquinas. Aunque hay más combinaciones.

Valores de radius

Definimos border, seguido de la esquina que vamos a modificar y por ultimo agregamos la palabra radius. Ej: border-top-left-radius, afectará solo a la esquina superior izquierda. El valor top-right cambiará la esquina superior derecha, de igual manera con bottom-left y bottom-right a las esquinas inferiores izquierda y derecha.

También podemos cambiar dos esquinas opuestas con los valores border-top-left-radius y border-bottom-right-radius que modificara las esquinas superior izquierda y la esquina inferior derecha. Lo mismo si usamos el valor top-right y bottom-left para hacerlo inversamente. Se puede usar la propiedad abreviada para obtener los mismos resultados.
